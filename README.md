# 30 Day SQL Challenge

**30 days. 30 lessons. From zero to SQL confident.**

Master SQL from scratch with daily video lessons, hands-on datasets, and real-world exercises  - all using PostgreSQL. Whether you're aiming for your first data role or levelling up your SQL skills, this challenge gives you the structure and practice to get there.

📺 [Watch the series on YouTube](https://www.youtube.com/@sdw-online)

---

## What You'll Learn

### Week 1: SQL Fundamentals (Days 1-7)
Build your foundation  - querying, filtering, sorting, aggregating, and modifying data.
> SELECT, WHERE, ORDER BY, GROUP BY, INSERT/UPDATE/DELETE, constraints, **Project 1**

### Week 2: Data Quality & Transformation (Days 8-14)
Handle messy data like a professional  - NULLs, functions, conditional logic, and reusable query patterns.
> NULLs, string/numeric/date functions, CASE WHEN, subqueries, CTEs, **Project 2**

### Week 3: JOINs & Relational Data (Days 15-21)
Connect tables and think relationally  - the skill that separates beginners from job-ready candidates.
> INNER/LEFT/RIGHT/FULL OUTER/CROSS/Self JOINs, UNION, normalisation, recursive CTEs, star schema, **Project 3**

### Week 4: Advanced Analytics & Performance (Days 22-30)
Write production-grade SQL  - window functions, slowly changing dimensions, views, metadata, indexing, and a full capstone project.
> Window functions, SCD/MERGE, views, information schema, UDFs, EXPLAIN/indexing, PostgreSQL pro tips, **Capstone Project**

## Setup Guide

### Step 1: Install PostgreSQL & pgAdmin
Follow this video to install PostgreSQL and pgAdmin on your machine:

👉 [Install PostgreSQL & pgAdmin (YouTube)](https://youtu.be/g8GwhsVPaOg)

### Step 2: Clone This Repo
```bash
git clone https://github.com/sdw-online/30-Day-SQL-Challenge.git
```

Or download as ZIP: **Code → Download ZIP** on this page.

### Step 3: Create Your Database
Open pgAdmin, connect to your PostgreSQL server, and run:

```sql
CREATE DATABASE sql_challenge;
```

### Step 4: Start Day 1
Go to the [`day-01/`](day-01/) folder, open the README, and follow along.

Each day has:
- A dataset (SQL to create tables and insert data)
- Exercises (questions to practice what you learnt)
- A link to the video lesson
